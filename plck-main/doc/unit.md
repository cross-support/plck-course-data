
# ユニット

PLCKでは、各講座をユニットとして取り扱います。
ユニットを複数作成することで、様々な講座を作成することができます。

ユニットは `contents` ディレクトリの `units` フォルダー内で管理します。

<pre>
contents
┣ units
    ┣ <ユニット名のフォルダ>
        ┣ plck.config.yaml
        ┣ lang 
            ┣ <言語コード>.yaml
</pre>

ユニットは、 `plck.config.yaml` ファイルと、言語ファイルで構成されます。<br />
`plck.config.yaml` では、ユニット全体の設定を定義します。<br />
複数言語に対応する場合、ユニットのタイトルなどの多言語対応が必要な箇所については `lang` ディレクトリに言語コードに対応したコンフィグファイルを指定する必要があります。例えば、英語にも対応する場合は、 `lang/en.yaml` という形で言語ファイルを追加します。


## ユニットを追加する

ユニットを追加する場合は、PLCKのコマンドラインを用いて作成することができます。

### `npm run add-unit <ユニット名>`

`npm run add-unit <ユニット名>`　を実行することで、ユニットを追加できます。<br />
コマンドプロンプトが表示され、以下の要素について設定を求められます。

### ユニットのタイトル
ユニットのタイトルを指定します。<br />
日本語タイトルや英語タイトルなど、デフォルトに指定したい言語でタイトルを入力してください。
何も入力しない場合は、日本語で「タイトル」が指定されます

### デフォルト言語

デフォルトの言語をリストから選択してください。

* 矢印キー上下でリストを移動することができます。

### その他言語

多言語対応でユニットを作成する場合は、その言語を指定してください。

* 矢印キー上下でリストを移動することができます
* <スペース>キーを押すと多言語で指定したい項目を選択・選択解除することができます。
* <エンター>キーを押すと入力を確定することができます。
* 何も指定せずにエンターを押すと、多言語指定しない状態でユニットを作成することができます。



## `plck.config.yaml`

PLCKの全体の設定を、 `plck.config.yaml` で設定することができます。<br />


`plck.config.yaml` の例
```yaml
title: 講義のタイトルを入力します 
scene_menu: on
complete_flg_scene: 1
threshold_pc_smartphone: 768
display_complete_alert: off
popup_explanation_text: このユニットは終了です。お疲れ様でした
popup_next_unit_text: 次のユニットへ
popup_close_text: 終了する
multi_lang: on
default_lang: ja
label_text: Choose your language
lang:
  - ja
  - en
frames:
  - main:
      name: video-1
      type: video
    sub:
      name: quiz-1
      type: quiz
```


| プロパティー名                                            | 概要                                                                                             | 値                       |
|----------------------------------------------------|------------------------------------------------------------------------------------------------|-------------------------|
| title                                              | 講座のタイトルを指定できます。                                                                                | `文字列（HTMLの可能）`          |
| scene_menu                                         | メニューを表示するかを指定します。 `off` に指定した場合、メニューを使ったシーンの選択および言語選択ができなくなりますのでお気をつけください。                     | `on` or `off`           |
| <a id="complete_flag_scene"></a>complete_flg_scene | 指定した番号に対応するシーンは、画面に表示すると完了状態になります。 完了状態にしたいフレームの番号を ,(カンマ)つなぎで記述します。<br />フレームの数値は1～で指定してください。 | `文字列` 例:（1,2,3）         |
| display_complete_alert                             | すべての講座が完了状態になったときに講座完了のポップアップを表示するかを指定できます。                                                    | `on` or `off`           |
| popup_explanation_text                             | `display_complete_alert` で表示できるポップアップの本文を指定できます。                                               | `文字列（HTMLの可能）`          |
| popup_next_unit_text                               | `display_complete_alert` で表示できるポップアップのユニットに移動するボタンの文字列を指定します。                                  | `文字列（HTMLの可能）`          |
| popup_close_text                                   | `display_complete_alert` で表示できるポップアップの終了するボタンの文字列を指定します。                                       | `文字列（HTMLの可能）`          |
| multi_lang                                         | 複数の言語の有効の有無を指定します。<br />有効となっている場合は、画面上に言語を選択するセレクトボックスが表示されます。                                | `on` もしくは `off`         |
| default_lang                                       | デフォルトで表示する言語を指定します                                                                             | [言語コード](./locale.md)    |
| label_text                                         | 言語選択をするセレクトボックスのタイトル文を指定します。                                                                   | `文字列（HTMLの可能）`          |
| lang                                               | この講座で使用できる言語を指定します。                                                                            | [言語コード](./locale.md)の配列 |
| frames                                             | 講座に設定するシーンをフレームとして指定します。 `frame` オブジェクトの配列についてはこちらを[参照](#frame)してください                          | `配列: frame[]`           |

## `frame` オブジェクト

`frame` オブジェクトを指定して、作成した各シーンの設定を行います。<br />
`frame` オブジェクトでは `main` と `sub` を指定することができ、最大2つのシーンを１つのフレームに表示することができます。

`main` と `sub` では以下のプロパティーを指定することができます。


| プロパティー名 | 概要       | 値 |
|---------|----------|----|
| name    | 作成したシーンの名前を指定します | `文字列`  |
| type    | シーンのタイプを指定します | `html / chat / video / slide / quiz`  |


```yaml
frames:
  - main:
      name: video-1
      type: video
    sub:
      name: quiz-1
      type: quiz
```
