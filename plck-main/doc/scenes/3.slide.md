
# slideシーン

スライドを表示することができるシーンです。


## `config.yaml` 

`config.*.yaml` で指定できるプロパティーは以下の通りです。


| プロパティー名 | 概要 | 値 |
|------|----|----|
| title | シーンのタイトルを指定します。<br />指定された文字列は、ヘッダーやメニューに表示されます。 | `文字列(HTMLも可能)` |
| btn_next_scene_name            | 次のシーンへへ移動するボタンのラベルを指定します。                               | `文字列`                          |
| btn_next_unit_name             | 次のユニットへ移動するボタンのラベルを指定します。                               | `文字列`                          |
| btn_end_unit_name              | ユニットを終了するボタンのラベルを指定します。                                 | `文字列`                          |

| btn_next_scene_name | 次のシーンへ行くボタンのラベルを指定します。 | `文字列（HTMLも可能）` |

| slide_speed | 次のスライドに遷移するトランジションのスピードを指定します | `数値` |
| img | 表示するスライドの画像URLを指定します。 | `配列(string[])` |

### `img`の指定方法

`img` プロパティーは配列で指定する必要があります。

```yaml
img:
# yamlの配列
    - 画像1.png
    - 画像2.png
    - 画像3.png
```

`scenes` ディレクトリから画像を参照したい場合は、 `@scenes/~` で画像を指定して読み込むことが可能です。この時に必ず"(クオーテーション)で文字列を囲って記述してください。

```yaml
img:
# yamlの配列
    # ""　で囲わないとエラーになります。
    - "@scenes/slide/img/slide-1.png"
    - "@scenes/slide/img/slide-2.png"
    - "@scenes/slide/img/slide-3.png"
```

## `customHook.js` で指定できるイベント

| イベントメソッド名 | 概要 | 引数 |
|------|----|----|
| `slideInitialized` | スライドが初期化された時（初めて表示される時）に実行されるイベントです。 | なし |
